🛒 Shop Smart – (Full-Stack MERN)

A modern shopping web application built using the MERN stack (MongoDB, Express, React, Node.js).
It allows users to browse products, manage cart, handle addresses, and simulate checkout seamlessly 🚀

🌟 Features
🧾 User Features
👤 User Authentication (Signup/Login with JWT)
❤️ Add to Favorites
🛒 Cart Management (Add/Remove/Update items)
📦 Address Management (Multiple delivery addresses)
🔍 Smart Search functionality
💳 Mock Payment system
🏪 Backend Features
📡 RESTful APIs for:
/api/products
/api/cart
/api/auth
/api/address
/api/search
🔐 JWT Authentication & bcrypt password hashing
🌍 MongoDB Atlas integration
📦 Product & data management
⚙️ Tech Stack
Frontend
React.js (Hooks + Context API)
Tailwind CSS
Axios
Backend
Node.js + Express.js
MongoDB Atlas
Mongoose
dotenv
bcryptjs + jsonwebtoken
📂 Folder Structure
/src → Frontend code
/public → Static assets
/Backend → Backend APIs & server
🚀 How to Run
Frontend

npm install
npm start

Backend

cd Backend
npm install
node server.js
